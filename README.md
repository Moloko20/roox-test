# roox-test

RooX Solutions employment test assignment - SPA React App.

## Installation

Clone the repository:
```bash
git clone https://github.com/Moloko20/roox-test
```

Use the npm to install packages.

```bash
npm i
```

## Usage

Run project:

```bach
npm run watch
```
